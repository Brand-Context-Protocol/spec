# Governance

## BDFL
Nick Strada (@nickstrada / Encoded Brands) serves as the Benevolent Dictator for Life (BDFL) for the Brand Context Protocol specification. He has final authority on all technical and protocol decisions.

## Independence
The BCP specification is independent of any single commercial entity. While Encoded Brands initially authored and maintains the protocol, the goal is to grow a neutral ecosystem of co-maintainers and eventually transition to a formal foundation or working group.

## Proposals (RFCs)
Changes to the specification follow a Request for Comments (RFC) process:
1. Open a GitHub Issue with the `rfc` label.
2. Discuss the proposal with the community.
3. Reach consensus or receive BDFL approval.
4. Merge the change into the specification.
