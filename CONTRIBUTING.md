# Contributing to BCP

We welcome contributions to the protocol, schemas, and documentation.

## How to Help
- **Report Bugs:** Open an issue for spec ambiguities or schema errors.
- **Propose Changes:** Use the RFC process detailed in [GOVERNANCE.md](GOVERNANCE.md).
- **Add Examples:** Submit a PR with a well-formatted `brand.md` example from a live domain.

## Code of Conduct
Please adhere to our [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
